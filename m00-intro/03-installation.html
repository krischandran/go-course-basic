Установка Go

Прежде чем писать код на Go, нужно установить компилятор. Это займет несколько минут - процесс простой и хорошо документированный.

УСТАНОВКА НА САМОМ ДЕЛЕ

Что мы устанавливаем

Мы устанавливаем компилятор Go - программу, которая будет запускать наш код.

i

Компилятор - это переводчик. Ты пишешь на языке Go, компилятор переводит в язык, который понимает компьютер.

Зависит от операционной системы

Установка отличается в зависимости от того, какую операционную систему ты используешь: Windows, Mac или Linux. Но в целом ничего сложного.

ПОШАГОВАЯ УСТАНОВКА

Шаг 1: Скачать Go

Заходим на официальный сайт: go.dev

Нажимаем кнопку Download. Откроется страница с версиями для разных систем:

Система Что выбрать
Windows Версия для Windows
Mac Две версии для Mac
Linux Версия для Linux

Выбирай версию для своей системы. Обычно подходит та, что предлагается по умолчанию. Можно выбрать специфическую версию, если у тебя особая архитектура.

Шаг 2: Установить

Windows:

  1. Запусти скачанный файл
  2. Нажимай "Далее" (Next) на каждом шаге
  3. После установки перезагрузи компьютер
!

Windows любит перезагрузку после установки. Без нее Go может не заработать.

Mac:

  1. Запусти скачанный файл
  2. Следуй инструкциям установщика
  3. Готово - все просто

Linux:

  1. Удали старую версию Go, если она была
  2. Распакуй архив правильно
  3. Пропиши путь в переменной PATH
i

PATH - это переменная окружения, которая говорит системе, где искать программы. Когда ты пишешь go version, система ищет программу go в папках, перечисленных в PATH. Если Go не в PATH, система его не найдет и скажет "команда не найдена".

На сайте go.dev есть подробные инструкции для каждой системы - Install instructions. Там все написано.

Шаг 3: Проверить установку

После установки нужно проверить, что все работает. Открой терминал и выполни команду.

i

Терминал - это программа для ввода текстовых команд. На Windows ищи "Командная строка" или "PowerShell" в меню Пуск. На Mac - приложение "Терминал" (Terminal). На Linux - обычно Ctrl+Alt+T.

Bash 
go version

Если все установлено правильно, ты увидишь установленную версию:

go version go1.23.2 darwin/amd64

Это значит: Go успешно установлен, никаких проблем не будет.

i

Если команда не найдена - проверь, перезагрузил ли ты компьютер (Windows) или правильно ли прописал PATH (Linux).

ПРО ВЕРСИИ

Версии обновляются часто

Новые версии Go выходят регулярно - буквально каждый месяц появляются обновления.

Рекомендация простая: всегда устанавливай новые версии Go. Тогда все будет работать хорошо.

ПРОВЕРЬ СЕБЯ

Вопросы

01 На каком сайте скачивается официальный Go?
Показать ответ

go.dev - официальный сайт языка Go. Там есть кнопка Download и инструкции по установке для всех операционных систем.

02 Какую команду нужно выполнить, чтобы проверить установку Go?
Показать ответ

go version

Эта команда показывает установленную версию Go. Если команда работает и показывает версию - Go установлен успешно.

03 Что нужно сделать после установки Go на Windows?
Показать ответ

Перезагрузить компьютер. Windows требует перезагрузку после установки.

04 Что такое PATH и зачем его настраивать при установке Go на Linux?
Показать ответ

PATH - это переменная окружения, которая говорит системе, где искать программы.

Без настройки PATH:

  • Система не знает, где находится Go
  • Команда go version вернет ошибку "команда не найдена"

После добавления пути к Go в PATH:

  • Система находит программу go в нужной папке
  • Все команды Go работают из любого места

Задачи

01 Установи Go
  1. Перейди на go.dev
  2. Скачай версию для своей системы
  3. Установи по инструкции
  4. Выполни go version в терминале
Показать решение

После успешной установки команда go version должна вывести что-то вроде:

go version go1.23.2 linux/amd64

Числа версии могут отличаться - это нормально. Главное, что команда работает и показывает версию.

02 Запиши свою версию

Выполни команду go version и запомни результат. Это пригодится для отладки, если что-то пойдет не так.

Показать решение

Пример вывода: go version go1.23.2 darwin/arm64

Здесь:

  • go1.23.2 - номер версии
  • darwin - система (Mac)
  • arm64 - архитектура процессора
