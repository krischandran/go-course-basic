Как устроена Go-программа

Go-проект - это не просто набор файлов. У него есть четкая структура: модуль, пакеты, зависимости. Понимание этой структуры - фундамент для работы с любым Go-проектом.

МОДУЛЬ: ЕДИНИЦА GO-ПРОЕКТА

Что такое модуль

Модуль - это единица кода в Go. Это папка с файлом go.mod, который описывает проект: его имя и зависимости.

До модулей (Go < 1.11) был GOPATH - глобальная папка для всего Go-кода. Это создавало проблемы с версиями библиотек. Модули решили эту проблему.

Создание модуля

Для полноценной работы с Go-проектом нужно инициализировать модуль. Это делается один раз в начале проекта.

Шаг 1. Создайте папку для проекта и откройте её в VS Code

Шаг 2. Откройте терминал: меню Terminal - New Terminal

Шаг 3. Выполните команду:

Bash 
go mod init your-go

Вместо your-go можно написать любое имя. Для реальных проектов часто используют путь репозитория:

Bash 
go mod init github.com/username/projectname

Результат: Появится файл go.mod:

go.mod 
module your-go

go 1.21
Строка Значение
module your-go Имя модуля (идентификатор проекта)
go 1.21 Версия Go для компиляции

Когда вы добавляете внешние библиотеки, они автоматически записываются в go.mod:

go.mod 
module your-go

go 1.21

require (
    github.com/some/library v1.2.3
)

СТРУКТУРА GO-ФАЙЛА

Каждый .go файл имеет строгую структуру из трех частей:

Структура .go файла
1 package main Объявление пакета
2 import "fmt" Импорты
3 func main() {...} Код
!

Порядок обязателен. Нельзя поставить import перед package или код перед import.

Часть 1: Объявление пакета

Go 
package main

Каждый Go-файл принадлежит какому-то пакету. Строка package main означает:

  • Это исполняемая программа (не библиотека)
  • Компилятор будет искать функцию main() как точку входа

Часть 2: Импорты

Go 
import "fmt"

Подключаем пакеты, которые будем использовать. В Go все разделено по пакетам:

  • fmt - форматированный ввод-вывод
  • strings - работа со строками
  • math - математические функции

Множественные импорты:

Go 
import (
    "fmt"
    "strings"
    "math"
)

Круглые скобки позволяют перечислить несколько пакетов.

Часть 3: Код

Go 
func main() {
    fmt.Println("Всем привет!")
}

Функция main - точка входа. С неё начинается выполнение программы.

ЗАПУСК ПРОГРАММЫ

Команда go run

Самый быстрый способ запустить программу:

Bash 
go run main.go

Команда go run:

  1. Компилирует файл
  2. Создает временный исполняемый файл
  3. Запускает его
  4. Удаляет временный файл

Вывод:

Output 
Всем привет!

Команда go build

Для создания исполняемого файла:

Bash 
go build -o myprogram main.go

Создаст файл myprogram (или myprogram.exe на Windows), который можно запускать без Go.

ВЫВОД ЧИСЕЛ И ВЫРАЖЕНИЙ

В предыдущем уроке мы выводили строки. fmt.Println также умеет выводить числа и вычислять выражения.

Вывод чисел

Go 
package main

import "fmt"

func main() {
    fmt.Println(26)           // 26
    fmt.Println(1024 + 48)    // 1072
    fmt.Println(5 + 8 * 2)    // 21
    fmt.Println((5 + 8) * 2)  // 26
}

Вывод:

Output 
26
1072
21
26

Операции выполняются по правилам математики: сначала умножение и деление, потом сложение и вычитание. Скобки меняют приоритет.

Вывод нескольких значений

Можно передать несколько аргументов через запятую:

Go 
package main

import "fmt"

func main() {
    fmt.Println("5 + 7 =", 12)      // 5 + 7 = 12
    fmt.Println(5, "+", 7, "=", 12) // 5 + 7 = 12
    fmt.Println(3, 6)               // 3 6
}

Вывод:

Output 
5 + 7 = 12
5 + 7 = 12
3 6

Go автоматически вставляет пробел между аргументами. Это удобно для вывода данных с пояснениями.

VS CODE: ПОЛЕЗНЫЕ ПРИЕМЫ

Просмотр документации

Наведите курсор на функцию - появится описание:

Go 
fmt.Println // навести курсор

Описание на английском, но можно скопировать и перевести.

Переход к определению

Зажмите Ctrl (Cmd на Mac) и кликните на функцию - откроется исходный код этой функции. Так можно изучать, как работают стандартные пакеты.

Горячие клавиши

Клавиши Действие
Ctrl+S Сохранить файл
Ctrl+P Быстрый поиск файла
Tab Добавить отступ
Shift+Tab Убрать отступ
i

Совет: Help - Keyboard Shortcuts Reference покажет все горячие клавиши VS Code.

Автоформатирование

При сохранении (Ctrl+S) Go-расширение автоматически форматирует код: выравнивает отступы, добавляет пустые строки. Не нужно следить за форматированием вручную.

ПРОВЕРЬ СЕБЯ

Вопросы

01 Зачем нужен файл go.mod?
Показать ответ

Файл go.mod определяет Go-модуль. Он содержит имя проекта, версию Go и список зависимостей. Без него проект не считается валидным модулем и могут возникнуть проблемы с импортами.

02 В каком порядке идут части Go-файла?
Показать ответ

1. Объявление пакета (package)

2. Импорты (import)

3. Код (функции, переменные)

Порядок строгий - нарушение вызовет ошибку компиляции.

03 Чем go run отличается от go build?
Показать ответ

go run компилирует и сразу запускает программу (временный файл удаляется). go build создает постоянный исполняемый файл, который можно запускать независимо от Go.

04 Что выведет fmt.Println(2 + 3 * 4)?
Показать ответ

Выведет 14. Сначала умножение (3 * 4 = 12), потом сложение (2 + 12 = 14). Математические приоритеты сохраняются.

Задачи

01 Что выведет?
Go 
package main

import "fmt"

func main() {
    fmt.Println(5 + 2*4 - 10)
}
Показать решение

Вывод: 3

Почему:

1. Сначала умножение: 2 * 4 = 8

2. Затем слева направо: 5 + 8 - 10 = 3

02 Что выведет?
Go 
package main

import "fmt"

func main() {
    fmt.Println((5 + 2) * 4 - 10)
}
Показать решение

Вывод: 18

Почему:

1. Скобки имеют приоритет: 5 + 2 = 7

2. Умножение: 7 * 4 = 28

3. Вычитание: 28 - 10 = 18

03 Что выведет?
Go 
package main

import "fmt"

func main() {
    fmt.Println("#", 2)
}
Показать решение

Вывод: # 2

Почему: Go автоматически вставляет пробел между аргументами в Println.

04 Что выведет?
Go 
package main

import "fmt"

func main() {
    fmt.Println(2, "#", 3, "#", 4, "#", 5)
}
Показать решение

Вывод: 2 # 3 # 4 # 5

Почему: Каждый аргумент выводится с пробелом между ними. Строки и числа чередуются.

05 Что выведет?
Go 
package main

import "fmt"

func main() {
    fmt.Print("5 + 7 = ", 12)
}
Показать решение

Вывод: 5 + 7 = 12

Почему:

1. Строка "5 + 7 = " содержит пробел в конце (перед закрывающей кавычкой)

2. К ней добавляется число 12

3. Print не добавляет автоматический пробел между строкой и числом, но в строке уже есть пробел

РЕЗЮМЕ

Key Ключевые идеи
  • Go-проект начинается с go mod init имя-модуля
  • Файл go.mod хранит имя модуля и зависимости
  • Go-файл имеет структуру: package - import - код
  • go run - компилирует и запускает, go build - создает исполняемый файл
  • fmt.Println умеет выводить числа и вычислять выражения
  • Несколько аргументов разделяются запятыми, Go добавляет пробелы

Главная ловушка: Забыть go mod init и получить ошибки импорта.

Правило: Новый проект = сразу go mod init.
