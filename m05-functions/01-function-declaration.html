Объявление функций

Функции позволяют избежать дублирования кода и разбить программу на логические части. Это основной инструмент структурирования программ в Go.

Зачем нужны функции

Представь, что нужно поприветствовать каждого пользователя курса:

Go 
package main

import "fmt"

func main() {
    var name1 string
    fmt.Scan(&name1)
    fmt.Println("Уважаемый", name1, ", добро пожаловать на курс Go!")
    fmt.Println("После курса ты будешь знать основы языка.")

    var name2 string
    fmt.Scan(&name2)
    fmt.Println("Уважаемый", name2, ", добро пожаловать на курс Go!")
    fmt.Println("После курса ты будешь знать основы языка.")

    var name3 string
    fmt.Scan(&name3)
    fmt.Println("Уважаемый", name3, ", добро пожаловать на курс Go!")
    fmt.Println("После курса ты будешь знать основы языка.")
}

Проблемы очевидны:

  • Дублирование кода — один и тот же блок повторяется трижды
  • Сложность изменений — если нужно изменить текст, придётся править в нескольких местах

Функция — это именованный блок кода, который можно вызывать многократно. Вместо копирования кода, мы выносим его в функцию и вызываем по имени.

Функции на самом деле

Как Go выполняет программу

Когда Go запускает программу, он начинает с функции main. Это точка входа — место, откуда начинается выполнение.

1. Go находит func main()
2. Выполняет код внутри main построчно
3. Когда встречает вызов функции — переходит в неё
4. Выполняет её код и возвращается обратно
5. Продолжает выполнение main

Функция — это именованный блок

Каждая функция имеет:

  • Имя — по нему функцию вызывают
  • Тело — код, который выполняется при вызове
  • Круглые скобки — обязательны даже если пустые
Go 
func printHello() {
    fmt.Println("Привет!")
}

Здесь printHello — имя функции, fmt.Println("Привет!") — тело функции.

Синтаксис объявления функции

Простейшая функция, которая ничего не принимает и ничего не возвращает:

Go 
func имяФункции() {
    // тело функции
}
Элемент Описание
func Ключевое слово для объявления функции
имяФункции Имя, по которому функцию вызывают
() Круглые скобки (пустые = ничего не принимает)
{} Фигурные скобки ограничивают тело функции

Пример

Go 
package main

import "fmt"

func main() {
    printMessage()  // вызов функции
    printMessage()  // можно вызывать многократно
}

func printMessage() {
    fmt.Println("Привет, ученики курса Go!")
}
Привет, ученики курса Go!
Привет, ученики курса Go!

Расположение функций

Функции объявляются вне main, обычно после неё:

Go 
package main

import "fmt"

func main() {
    // main — главная функция, точка входа
    printMessage()
}

// Дополнительные функции — после main
func printMessage() {
    fmt.Println("Привет!")
}
!

Функции нельзя объявлять внутри других функций. Каждая функция — на верхнем уровне файла.

Почему можно вызвать функцию до её объявления?

Go сначала читает весь файл, находит все функции, и только потом начинает выполнение с main. Поэтому порядок объявления функций в файле не важен.

Правила именования функций

1. Начинается с глагола — функция выполняет действие:

  • printMessage — выводит сообщение
  • calculateSum — вычисляет сумму
  • getUserInput — получает ввод пользователя

2. Начинается с маленькой буквы — для функций внутри пакета

3. camelCase — каждое следующее слово с большой буквы:

Go 
// Хорошие имена
func printGreeting() { ... }
func calculateArea() { ... }
func readUserName() { ... }

// Плохие имена
func message() { ... }        // нет глагола
func Print_Message() { ... }  // подчёркивание, большая буква
func pm() { ... }             // непонятно что делает

Вызов функции

Чтобы выполнить код функции, её нужно вызвать. Вызов — это имя функции с круглыми скобками:

Go 
printMessage()  // вызов функции

Ловушка: забыли скобки

Go 
func main() {
    printMessage   // Это НЕ вызов! Просто имя функции
}

Без скобок функция не выполнится.

Ловушка: объявили, но не вызвали

Go 
func main() {
    // Забыли вызвать printMessage()
}

func printMessage() {
    fmt.Println("Привет!")  // Никогда не выполнится
}

Функция существует, но без вызова её код не выполняется.

Четыре вида функций

В Go функции делятся на 4 типа в зависимости от того, принимают ли они данные и возвращают ли результат:

1. Ничего не принимает, ничего не возвращает: func doSomething() { }
2. Принимает данные, ничего не возвращает: func printName(name string) { }
3. Ничего не принимает, возвращает результат: func getMessage() string { }
4. Принимает данные, возвращает результат: func add(a, b int) int { }

В этом уроке мы рассмотрели первый вид. Остальные — в следующих уроках.

Проверь себя

Вопросы

01 Что такое функция в Go?
Показать ответ

Функция — это именованный блок кода, который можно вызывать по имени. Функции позволяют избежать дублирования кода и разбить программу на логические части.

02 Какая функция является точкой входа в программу Go?
Показать ответ

Функция main в пакете main. С неё начинается выполнение любой Go-программы.

03 Можно ли объявить функцию внутри другой функции?
Показать ответ

Нет, в Go функции объявляются только на верхнем уровне файла. Нельзя объявить функцию внутри main или другой функции.

04 Почему имя функции должно начинаться с глагола?
Показать ответ

Функция выполняет действие. Глагол в начале имени сразу говорит, что делает функция: printMessage — выводит сообщение, calculateSum — вычисляет сумму.

Задачи

01 Что выведет код? 
package main

import "fmt"

func main() {
    fmt.Println("Начало")
    printHello()
    fmt.Println("Конец")
}

func printHello() {
    fmt.Println("Привет!")
}
Показать решение

Вывод:

Начало
Привет!
Конец

Разбор: Выполняется main, выводится "Начало", вызывается printHello() — выводится "Привет!", возвращаемся в main — выводится "Конец".

02 Что выведет код? 
package main

import "fmt"

func main() {
    printA()
    printA()
    printB()
}

func printA() {
    fmt.Print("A")
}

func printB() {
    fmt.Print("B")
}
Показать решение

Вывод: AAB

Разбор: Первый вызов printA() выводит "A", второй — "A", вызов printB() выводит "B".

03 Найди ошибки 
package main

import "fmt"

func main() {
    PrintMessage

    func printAnother() {
        fmt.Println("Другое сообщение")
    }
}

func PrintMessage() {
    fmt.Println("Привет!")
}
Показать решение

Ошибки:

  1. PrintMessage вызывается без скобок — нужно PrintMessage()
  2. Функция printAnother объявлена внутри main — нужно вынести наружу
  3. PrintMessage начинается с большой буквы — для внутренних функций используют маленькую
04 Сколько раз выполнится тело функции? 
package main

import "fmt"

func main() {
    for i := 0; i < 3; i++ {
        printHello()
    }
}

func printHello() {
    fmt.Println("Привет!")
}
Показать решение

Ответ: 3 раза

Разбор: Цикл выполняется 3 итерации (i = 0, 1, 2). На каждой итерации вызывается printHello(), поэтому тело функции выполнится 3 раза.

Резюме Ключевые идеи
  • Функция — именованный блок кода для повторного использования
  • Объявление: func имя() { тело }
  • Вызов: имя() — обязательно со скобками
  • Функции объявляются вне main, обычно после неё

Правило: Функция выполняется только когда её вызывают. Без вызова код функции не работает.
