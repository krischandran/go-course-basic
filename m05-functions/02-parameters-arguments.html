Параметры и аргументы

Параметры позволяют передавать данные в функцию извне, делая её гибкой и переиспользуемой для разных входных значений.

Ограничение простых функций

В прошлом уроке функция printGreeting сама читала имя пользователя:

Go 
func printGreeting() {
    var name string
    fmt.Scan(&name)
    fmt.Println("Привет,", name)
}

Проблема: имя пользователя застревает внутри функции. Мы не можем использовать его в main или передать уже известное имя.

Решение — параметры. Они позволяют передавать данные в функцию извне.

Параметры на самом деле

Что такое параметр

Параметр — это переменная, которая объявляется в круглых скобках функции и получает значение при вызове.

name — имя параметра
string — тип параметра
Параметр = переменная внутри функции, значение которой задаётся при вызове

Что такое аргумент

Аргумент — это конкретное значение, которое передаётся в функцию при вызове.

Go 
printName("Алексей")  // "Алексей" — аргумент

Параметр vs Аргумент

Термин Где Что это
Параметр В объявлении функции Переменная, ожидающая значение
Аргумент При вызове функции Конкретное значение

Синтаксис функции с параметрами

Go 
func имяФункции(параметр1 тип1, параметр2 тип2) {
    // тело функции
}

Один параметр

Go 
package main

import "fmt"

func main() {
    printGreeting("Алексей")
    printGreeting("Мария")
}

func printGreeting(name string) {
    fmt.Println("Привет,", name, "!")
}
Привет, Алексей !
Привет, Мария !

Несколько параметров

Go 
package main

import "fmt"

func main() {
    printUserInfo("Алексей", 25)
}

func printUserInfo(name string, age int) {
    fmt.Println("Имя:", name)
    fmt.Println("Возраст:", age)
}
Имя: Алексей
Возраст: 25

Сокращённая запись для одинаковых типов

Go 
// Полная запись
func printFullName(firstName string, lastName string) {
    fmt.Println(firstName, lastName)
}

// Сокращённая запись
func printFullName(firstName, lastName string) {
    fmt.Println(firstName, lastName)
}

Порядок аргументов

При вызове аргументы передаются в том же порядке, в каком объявлены параметры.

Ловушка: перепутанный порядок

Go 
firstName := "Иван"
lastName := "Иванов"

// Неправильный порядок!
printFullName(lastName, firstName)

Компилятор не поймёт ошибку — типы совпадают. Логика программы будет неверной.

!

Всегда проверяйте порядок аргументов при вызове функции. Компилятор проверяет только типы, но не смысл.

Типы аргументов должны совпадать

Go строго проверяет типы. Тип аргумента должен совпадать с типом параметра:

Go 
func printAge(age int) {
    fmt.Println("Возраст:", age)
}

func main() {
    printAge(25)      // OK: int
    printAge("25")    // Ошибка компиляции: string != int
}

Передача по значению

Для базовых типов (int, string, bool, float64) Go передаёт копию значения в функцию. Изменение параметра внутри функции не влияет на оригинал:

Go 
package main

import "fmt"

func main() {
    x := 10
    fmt.Println("До вызова:", x)

    tryToChange(x)

    fmt.Println("После вызова:", x)
}

func tryToChange(value int) {
    value = 100
    fmt.Println("Внутри функции:", value)
}
До вызова: 10
Внутри функции: 100
После вызова: 10

Функция изменила свою копию value, но оригинальная переменная x осталась неизменной.

Проверь себя

Вопросы

01 Чем отличается параметр от аргумента?
Показать ответ

Параметр — переменная в объявлении функции, ожидающая значение. Аргумент — конкретное значение, передаваемое при вызове функции.

02 Что произойдёт, если передать аргумент не того типа?
Показать ответ

Ошибка компиляции. Go строго проверяет типы — тип аргумента должен точно соответствовать типу параметра. Автоматического преобразования нет.

03 Изменится ли значение переменной в main, если изменить параметр внутри функции?
Показать ответ

Нет. Для базовых типов передаётся копия значения. Изменение копии внутри функции не влияет на оригинал.

Задачи

01 Что выведет код? 
package main

import "fmt"

func main() {
    printSum(3, 5)
}

func printSum(a, b int) {
    fmt.Println(a + b)
}
Показать решение

Вывод: 8

a получает значение 3, b получает значение 5. Выводится 3 + 5 = 8.

02 Что выведет код? 
package main

import "fmt"

func main() {
    x := 10
    double(x)
    fmt.Println(x)
}

func double(n int) {
    n = n * 2
}
Показать решение

Вывод: 10

Передаётся копия значения 10. Внутри функции n становится 20, но оригинальный x остаётся 10.

03 Что выведет код? 
package main

import "fmt"

func main() {
    a := "X"
    b := "Y"
    printTwo(b, a)
}

func printTwo(first, second string) {
    fmt.Print(first, second)
}
Показать решение

Вывод: YX

Вызывается printTwo(b, a) — передаём "Y" и "X". first = "Y", second = "X".

Key Ключевые идеи
  • Параметр — переменная в объявлении функции
  • Аргумент — значение, передаваемое при вызове
  • Порядок аргументов должен совпадать с порядком параметров
  • Базовые типы передаются по значению (копируются)

Правило: Количество и типы аргументов должны точно соответствовать параметрам функции.
