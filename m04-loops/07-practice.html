Задачи для закрепления

01 Сумма чисел от 1 до N

Напиши программу, которая читает число N и выводит сумму всех чисел от 1 до N.

Ввод: 5
Вывод: 15
Показать решение
Go 
package main

import "fmt"

func main() {
    var n int
    fmt.Scan(&n)

    sum := 0
    for i := 1; i <= n; i++ {
        sum = sum + i
    }

    fmt.Println(sum)
}

Разбор: Классический цикл, накопление суммы в переменной sum.

03 Четные числа

Напиши программу, которая выводит все четные числа от 2 до N.

Ввод: 10
Вывод: 2 4 6 8 10
Показать решение

Вариант 1: с проверкой

Go 
package main

import "fmt"

func main() {
    var n int
    fmt.Scan(&n)

    for i := 1; i <= n; i++ {
        if i%2 == 0 {
            fmt.Print(i, " ")

}

Вариант 2: с шагом 2 (эффективнее)

Go 
package main

import "fmt"

func main() {
    var n int
    fmt.Scan(&n)

    for i := 2; i <= n; i += 2 {
        fmt.Print(i, " ")
    }
}
04 Обратный отсчет

Напиши программу обратного отсчета от N до 1, затем выведи "Пуск!".

Ввод: 5
Вывод:
5
4
3
2
1
Пуск!
Показать решение
Go 
package main

import "fmt"

func main() {
    var n int
    fmt.Scan(&n)

    for i := n; i >= 1; i-- {
        fmt.Println(i)
    }

    fmt.Println("Пуск!")
}

06 Сумма цифр

Найди сумму цифр числа N.

Ввод: 12345
Вывод: 15
Показать решение
Go 
package main

import "fmt"

func main() {
    var n int
    fmt.Scan(&n)

    sum := 0
    for n > 0 {
        digit := n % 10  // последняя цифра
        sum = sum + digit
        n = n / 10       // отбрасываем последнюю цифру
    }

    fmt.Println(sum)
}

Разбор:

  • n % 10 дает последнюю цифру
  • n / 10 отбрасывает последнюю цифру
  • Повторяем, пока n > 0

08 Простое число

Определи, является ли число N простым (делится только на 1 и себя).

Ввод: 7 -> Вывод: Простое
Ввод: 12 -> Вывод: Составное
Показать решение
Go 
package main

import "fmt"

func main() {
    var n int
    fmt.Scan(&n)

    if n < 2 {
        fmt.Println("Составное")
        return
    }

    isPrime := true
    for i := 2; i < n; i++ {
        if n%i == 0 {
            isPrime = false
            break
        }
    }

    if isPrime {
        fmt.Println("Простое")
    } else {
        fmt.Println("Составное")
    }
}

Разбор: Проверяем делители от 2 до n-1. Если найден - число составное.

11 Числа Фибоначчи

Выведи первые N чисел Фибоначчи.

Числа Фибоначчи: 1, 1, 2, 3, 5, 8, 13, ... (каждое число - сумма двух предыдущих)

Ввод: 7
Вывод: 1 1 2 3 5 8 13
Показать решение
Go 
package main

import "fmt"

func main() {
    var n int
    fmt.Scan(&n)

    a := 1
    b := 1

    for i := 0; i < n; i++ {
        fmt.Print(a, " ")
        next := a + b
        a = b
        b = next
    }
}

Разбор: Храним два последних числа, на каждом шаге вычисляем следующее.

12 Угадай число с подсказками

Программа загадывает число от 1 до 100. Пользователь угадывает, программа подсказывает "больше" или "меньше".

Показать решение
Go 
package main

import "fmt"

func main() {
    secret := 42  // загаданное число
    attempts := 0

    for {
        var guess int
        fmt.Print("Ваша попытка: ")
        fmt.Scan(&guess)
        attempts++

        if guess == secret {
            fmt.Println("Угадали за", attempts, "попыток!")
            break
        } else if guess < secret {
            fmt.Println("Больше!")
        } else {
            fmt.Println("Меньше!")
        }
    }
}
13 Поиск в матрице*

Найди позицию первого вхождения числа T в "матрице" (два вложенных цикла до N).

Ввод: N=5, T=12
Вывод: 3 4
(потому что 3 * 4 = 12)
Показать решение
Go 
package main

import "fmt"

func main() {
    var n, t int
    fmt.Scan(&n, &t)

    found := false

outer:
    for i := 1; i <= n; i++ {
        for j := 1; j <= n; j++ {
            if i*j == t {
                fmt.Println(i, j)
                found = true
                break outer
            }
        }
    }

    if !found {
        fmt.Println("Не найдено")
    }
}

Резюме модуля

Key Что мы изучили
  • Классический for - для фиксированного числа повторений
  • for с условием - когда не знаем сколько итераций
  • Бесконечный цикл - for { } с обязательным выходом
  • break и continue - управление потоком цикла
  • Вложенные циклы - циклы внутри циклов
  • Метки - для выхода из нескольких циклов
Rules Главные правила
  • Если знаешь сколько раз - классический for
  • Если зависит от условия - for условие
  • Бесконечный цикл = обязательный break
  • break прерывает только ближайший цикл
  • Переменные вложенных циклов должны иметь разные имена
