Вещественные числа

Go предлагает два типа: float32 и float64. По умолчанию Go выбирает float64 - он точнее. В отличие от целых чисел, вещественные хранятся приблизительно, поэтому 0.1 + 0.2 не равно 0.3.

Тип float

Для чисел с дробной частью в Go есть тип float (от английского floating point - плавающая точка). Название связано с тем, как число хранится в памяти - точка может "плавать" между цифрами.

Go
package main

import "fmt"

func main() {
    price := 199.99
    fmt.Println(price)
}
Output
199.99

Переменная price автоматически получила тип float64. Это тип по умолчанию для дробных чисел в Go.

В Go есть два типа для дробных чисел:

ТипРазмерТочность
float324 байта~7 цифр
float648 байт~15 цифр

Если тип не указан явно, Go выбирает float64 - он точнее:

Go
package main

import "fmt"

func main() {
    a := 0.5            // float64 (по умолчанию)
    var b float32 = 0.5 // явно указан float32

    fmt.Println(a, b)
}
Output
0.5 0.5

Важно: в Go разделитель дробной части - точка, не запятая. Это отличается от математики, где пишут 3,14, а в коде нужно писать 3.14.

Почему два типа, а не один? Дело в балансе между точностью и памятью. float32 занимает в два раза меньше места, но и точность у него ниже. Для большинства задач используй float64 - памяти сейчас достаточно, а точность важнее.

Арифметика с float

С вещественными числами работают те же операции, что и с целыми:

Go
package main

import "fmt"

func main() {
    a := 5.0
    b := 2.0

    fmt.Println(a + b)  // сложение
    fmt.Println(a - b)  // вычитание
    fmt.Println(a * b)  // умножение
    fmt.Println(a / b)  // деление
}
Output
7
3
10
2.5

Главное отличие от целых чисел - деление. При делении двух целых чисел результат тоже целое:

Go
fmt.Println(5 / 2)   // 2 (целочисленное деление!)
fmt.Println(5.0 / 2) // 2.5 (вещественное деление)

Чтобы получить дробный результат, хотя бы одно из чисел должно быть float.

Ловушка: погрешность

У float типов есть неприятная особенность - они "приблизительные". Посмотри:

Go
package main

import "fmt"

func main() {
    f := 0.1
    f += 0.2
    fmt.Println(f)
}
Output
0.30000000000000004

Вместо ровных 0.3 получаем число с "хвостом"!

Это не баг Go - так работают float во всех языках программирования. Причина в том, как компьютер хранит дробные числа в двоичной системе. Некоторые десятичные дроби (например, 0.1) невозможно точно представить в двоичном виде - получается бесконечная дробь, которую приходится округлять.

Ещё пример:

Go
package main

import "fmt"

func main() {
    f := 5.8
    fmt.Println(f - 5.0)
}
Output
0.7999999999999998

Это фундаментальное ограничение представления чисел в компьютере. Чем больше целая часть числа, тем больше может быть погрешность.

!

Не используй float для денег! Для финансовых расчётов используют целые числа (копейки вместо рублей) или специальные библиотеки для точных вычислений.

Пакет math

Go предоставляет пакет math с математическими функциями. Вот самые полезные:

ФункцияДействие
math.Abs(x)Модуль числа
math.Sqrt(x)Квадратный корень
math.Pow(x, n)Возведение в степень
math.Max(a, b)Максимум из двух
math.Min(a, b)Минимум из двух
Go
package main

import (
    "fmt"
    "math"
)

func main() {
    fmt.Println(math.Abs(-9))     // 9
    fmt.Println(math.Sqrt(16))    // 4
    fmt.Println(math.Pow(2, 10))  // 1024
}
Output
9
4
1024

Важно: функции из math работают только с float64. Если передать int, будет ошибка компиляции.

Go
var n int = 16
// math.Sqrt(n)  // ошибка! нужен float64
math.Sqrt(float64(n))  // OK

Встроенные min и max

Начиная с Go 1.21 появились встроенные функции min и max, которые работают без импорта math:

Go
package main

import "fmt"

func main() {
    fmt.Println(max(8, 4))     // 8
    fmt.Println(min(8, 4))     // 4
    fmt.Println(max(3.5, 7.2)) // 7.2
}
Output
8
4
7.2

Эти функции удобнее - работают и с int, и с float, без приведения типов.

Нулевое значение

Если объявить переменную float без начального значения, она получит значение 0:

Go
package main

import "fmt"

func main() {
    var price float64
    fmt.Println(price)
}
Output
0

Почему float неточен?

Давай разберёмся, откуда берётся погрешность.

Компьютер хранит числа в двоичной системе. Число 0.5 в двоичной системе - это ровно 0.1 (одна вторая). А вот 0.1 в десятичной системе превращается в бесконечную двоичную дробь:

Text
0.1 (десятичное) = 0.0001100110011... (двоичное, бесконечно)

Компьютеру приходится обрезать эту дробь, и получается небольшая погрешность. При сложении погрешности накапливаются.

Это не проблема Go - так работает IEEE 754 (международный стандарт представления чисел с плавающей точкой), который используют все современные процессоры и языки программирования.

01Какой тип получит переменная x в коде x := 3.14?

float64 - Go по умолчанию выбирает float64 для дробных чисел.

Показать ответ

5 / 2 = 2 (целочисленное деление), 5.0 / 2 = 2.5 (вещественное).

Показать ответ

Из-за погрешности представления десятичных дробей в двоичной системе.

04Какое нулевое значение у типа float64?
Показать ответ

0 - нулевое значение для всех числовых типов.

05Можно ли использовать math.Sqrt с переменной типа int?
Показать ответ

Нет напрямую. Нужно преобразовать: math.Sqrt(float64(n)).

a := 7.0

fmt.Println(a / b)

3.5 - деление двух float даёт точный результат.

Задачи

01Площадь круга

Вычисли площадь круга с радиусом 5. Используй формулу S = π × r². Для π используй значение 3.14.

Показать решение
Go
package main

import "fmt"

func main() {
    r := 5.0
    pi := 3.14
    s := pi * r * r
    fmt.Println(s)
}

Результат: 78.5

02Среднее арифметическое

Найди среднее арифметическое чисел 10 и 15.

Показать решение
Go
package main

import "fmt"

func main() {
    a := 10.0
    b := 15.0
    avg := (a + b) / 2
    fmt.Println(avg)
}

Результат: 12.5

03Перевод температуры

Переведи 100 градусов Фаренгейта в Цельсии. Формула: C = 5/9 × (F - 32).

Показать решение
Go
package main

import "fmt"

func main() {
    f := 100.0
    c := 5.0 / 9.0 * (f - 32)
    fmt.Println(c)
}

Результат: 37.77777777777778

04Дробная часть

Дано число 7.85. Выведи его дробную часть (0.85).

Показать решение
Go
package main

import "fmt"

func main() {
    x := 7.85
    frac := x - float64(int(x))
    fmt.Println(frac)
}

Результат: 0.8500000000000005 (с погрешностью!)

05Гипотенуза

Найди гипотенузу прямоугольного треугольника с катетами 3 и 4. Используй теорему Пифагора и math.Sqrt.

Показать решение
Go
package main

import (
    "fmt"
    "math"
)

func main() {
    a := 3.0
    b := 4.0
    c := math.Sqrt(a*a + b*b)
    fmt.Println(c)
}

Результат: 5

06Расстояние между точками

Найди расстояние между точками A(1, 2) и B(4, 6). Формула: d = √((x2-x1)² + (y2-y1)²).

Показать решение
Go
package main

import (
    "fmt"
    "math"
)

func main() {
    x1, y1 := 1.0, 2.0
    x2, y2 := 4.0, 6.0
    d := math.Sqrt(math.Pow(x2-x1, 2) + math.Pow(y2-y1, 2))
    fmt.Println(d)
}

Результат: 5

  • Пакет math содержит Abs, Sqrt, Pow, Max, Min
