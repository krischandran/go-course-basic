Преобразование типов

Преобразование типов (type conversion) - это явное превращение значения одного типа в другой. Go не выполняет автоматические преобразования: нельзя присвоить int переменной int64 без явного указания. Синтаксис: тип(значение).

Преобразование между числовыми типами

Синтаксис простой - пишешь нужный тип и в скобках значение:

Go
package main

import "fmt"

func main() {
    num := 200         // тип int
    var num64 int64

    num64 = int64(num) // явное преобразование
    fmt.Println(num64) // 200
}
Output
200

Без преобразования Go выдаст ошибку: "cannot use num (type int) as type int64".

Преобразование float в int

При преобразовании вещественного числа в целое дробная часть отбрасывается (не округляется!):

Go
package main

import "fmt"

func main() {
    price := 199.99
    rounded := int(price)
    fmt.Println(rounded) // 199
}
Output
199

Обратное преобразование всегда безопасно:

Go
package main

import "fmt"

func main() {
    count := 7
    average := float64(count) / 2
    fmt.Println(average) // 3.5
}
Output
3.5

Переполнение при преобразовании

При преобразовании большого типа в маленький может произойти переполнение:

Go
package main

import "fmt"

func main() {
    num64 := int64(200)
    num8 := int8(num64)  // 200 не влезает в int8!
    fmt.Println(num8)    // -56
}
Output
-56

Go не предупреждает об этом - следить должен программист.

Символ по коду: string()

Функция string() с числом возвращает символ с этим кодом Unicode:

Go
package main

import "fmt"

func main() {
    fmt.Println(string(65))   // A
    fmt.Println(string(1055)) // П
}
Output
A
П

Код символа

Чтобы получить код символа, используй одинарные кавычки:

Go
package main

import "fmt"

func main() {
    fmt.Println('A')  // 65
    fmt.Println('П')  // 1055
    fmt.Println('0')  // 48 (код символа '0', не число 0!)
}
Output
65
1055
48

Строка в байты и руны

Для английского текста подходит []byte():

Go
package main

import "fmt"

func main() {
    s := "Hi"
    bytes := []byte(s)
    fmt.Println(bytes) // [72 105]
}
Output
[72 105]

Для любых символов используй []rune():

Go
package main

import "fmt"

func main() {
    s := "Привет"
    runes := []rune(s)
    fmt.Println(len(runes)) // 6 - по одному на букву
}
Output
6

Число и строка: strconv

strconv.Atoi - строка в число

Go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    s := "42"
    n, _ := strconv.Atoi(s)

    fmt.Println(n + 1) // 43
}
Output
43

strconv.Itoa - число в строку

Go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    n := 42
    s := strconv.Itoa(n)

    fmt.Println("Ответ: " + s) // Ответ: 42
}
Output
Ответ: 42

Важно не путать!

Go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    n := 65

    fmt.Println(string(n))       // A (символ с кодом 65!)
    fmt.Println(strconv.Itoa(n)) // 65 (строка "65")
}
Output
A
65

Проверь себя

Вопросы

01Можно ли присвоить int переменной int64 напрямую?
Показать ответ

Нет. Нужно явное преобразование: int64(num).

02Чему равно int(9.9)?
Показать ответ

9 - дробная часть отбрасывается, не округляется.

03Что произойдёт при int8(200)?
Показать ответ

Переполнение: результат будет -56. Go не предупреждает об этом.

04Чем отличается string(65) от strconv.Itoa(65)?
Показать ответ

string(65) возвращает символ с кодом 65 - букву "A". strconv.Itoa(65) возвращает строку "65".

05Почему []byte("Привет") содержит 12 элементов?
Показать ответ

[]byte разбивает строку на байты. Русская буква занимает 2 байта, поэтому 6 букв = 12 байтов.

06Какая функция преобразует строку "100" в число?
Показать ответ

strconv.Atoi(str) - возвращает число и ошибку.

-128 - переполнение int8.

Резюме

KeyПреобразование типов
  • Go требует явного преобразования типов - тип(значение)
  • При float -> int дробная часть отбрасывается
  • При преобразовании в меньший тип возможно переполнение
  • string(код) - символ по коду Unicode
  • strconv.Atoi(s) - строка в int
  • strconv.Itoa(n) - int в строку
  • []rune(s) - для любого текста (Unicode)
