bufio.Scanner -- чтение строк

В прошлом уроке мы познакомились с fmt.Scan. Но у него есть ограничение -- он разделяет ввод по пробелам. Если пользователь введет "Павел Тарасов", fmt.Scan прочитает только "Павел".

bufio.Scanner решает эту проблему -- он читает всю строку целиком, включая пробелы.

Базовое использование

Go 
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    scanner := bufio.NewScanner(os.Stdin)

    fmt.Print("Введите полное имя: ")
    scanner.Scan()
    fullName := scanner.Text()

    fmt.Println("Привет,", fullName)
}

Пример работы:

Введите полное имя: Павел Тарасов
Привет, Павел Тарасов

Как это работает

scanner := bufio.NewScanner(os.Stdin) -- создает сканер, который читает из стандартного ввода
scanner.Scan() -- читает одну строку (до нажатия Enter). Возвращает true если успешно, false если конец ввода
scanner.Text() -- возвращает прочитанную строку (без символа новой строки)

Чтение нескольких строк подряд

Каждый вызов scanner.Scan() читает следующую строку:

Go 
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    scanner := bufio.NewScanner(os.Stdin)

    fmt.Print("Имя: ")
    scanner.Scan()
    name := scanner.Text()

    fmt.Print("Город: ")
    scanner.Scan()
    city := scanner.Text()

    fmt.Print("Профессия: ")
    scanner.Scan()
    job := scanner.Text()

    fmt.Println("---")
    fmt.Println("Имя:", name)
    fmt.Println("Город:", city)
    fmt.Println("Профессия:", job)
}

Пример работы:

Имя: Анна Петрова
Город: Санкт-Петербург
Профессия: Go разработчик
---
Имя: Анна Петрова
Город: Санкт-Петербург
Профессия: Go разработчик

Один сканер на всю программу

!
Important

Создавай сканер один раз в начале программы. Не создавай новый сканер для каждого чтения.

Go 
// Правильно — один сканер
func main() {
    scanner := bufio.NewScanner(os.Stdin)

    scanner.Scan()
    name := scanner.Text()

    scanner.Scan()
    city := scanner.Text()
}

// Неправильно — много сканеров
func main() {
    scanner1 := bufio.NewScanner(os.Stdin)
    scanner1.Scan()
    name := scanner1.Text()

    scanner2 := bufio.NewScanner(os.Stdin)  // проблема!
    scanner2.Scan()
    city := scanner2.Text()
}

Почему? Каждый сканер имеет свой внутренний буфер. Создание нового сканера может "потерять" часть уже прочитанных данных.

Scan vs bufio.Scanner

Задача Инструмент
Прочитать одно слово fmt.Scan
Прочитать число fmt.Scan
Прочитать несколько значений через пробел fmt.Scan
Прочитать строку с пробелами bufio.Scanner
Прочитать несколько строк подряд bufio.Scanner

Tip

Проверь себя

Вопросы

01 В чем разница между fmt.Scan и bufio.Scanner?
Показать ответ

fmt.Scan разделяет ввод по пробелам -- каждое слово отдельно. bufio.Scanner читает всю строку целиком, включая пробелы.

Для ввода "Павел Тарасов":

  • fmt.Scan прочитает только "Павел"
  • bufio.Scanner прочитает "Павел Тарасов"
02 Почему нужно создавать один сканер на всю программу?
Показать ответ

Каждый сканер имеет внутренний буфер. При создании нового сканера часть уже прочитанных данных может быть потеряна -- они остались в буфере старого сканера.

03 Что делает метод scanner.Text()?
Показать ответ

Возвращает последнюю прочитанную строку (ту, которую прочитал предыдущий вызов scanner.Scan()). Строка возвращается без символа новой строки.

Задачи

01 Что выведет?
Go 
scanner := bufio.NewScanner(os.Stdin)
scanner.Scan()
a := scanner.Text()
scanner.Scan()
b := scanner.Text()
fmt.Println(b, a)

Ввод (две строки):

Привет
Мир
Показать решение

Вывод: Мир Привет

Почему: Первый Scan() читает "Привет" в a, второй читает "Мир" в b. Выводим в обратном порядке: сначала b, потом a.

02 Анкета

Напиши программу, которая спрашивает имя, возраст и город, затем выводит анкету.

Пример:

Имя: Иван Иванов
Возраст: 25
Город: Москва

=== Анкета ===
Имя: Иван Иванов
Возраст: 25
Город: Москва
Показать решение
Go 
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    scanner := bufio.NewScanner(os.Stdin)

    fmt.Print("Имя: ")
    scanner.Scan()
    name := scanner.Text()

    fmt.Print("Возраст: ")
    scanner.Scan()
    age := scanner.Text()

    fmt.Print("Город: ")
    scanner.Scan()
    city := scanner.Text()

    fmt.Println()
    fmt.Println("=== Анкета ===")
    fmt.Println("Имя:", name)
    fmt.Println("Возраст:", age)
    fmt.Println("Город:", city)
}
03 Найди ошибку
Go 
scanner1 := bufio.NewScanner(os.Stdin)
scanner1.Scan()
line1 := scanner1.Text()

scanner2 := bufio.NewScanner(os.Stdin)
scanner2.Scan()
line2 := scanner2.Text()

fmt.Println(line1, line2)
Показать решение

Ошибка: Создается два сканера для одного источника (os.Stdin).

Проблема: Данные могут потеряться между сканерами из-за буферизации.

Правильно:

Go 
scanner := bufio.NewScanner(os.Stdin)

scanner.Scan()
line1 := scanner.Text()

scanner.Scan()
line2 := scanner.Text()

fmt.Println(line1, line2)

Резюме

Key bufio.Scanner
  • bufio.Scanner читает строки целиком, включая пробелы
  • scanner.Scan() читает одну строку
  • scanner.Text() возвращает прочитанную строку
  • Создавай один сканер на всю программу
  • Используй fmt.Scan для отдельных слов/чисел, bufio.Scanner для полных строк
