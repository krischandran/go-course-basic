Print и Println в Go

Вывод данных -- первое, что осваивает любой программист. Разберем базовые функции Print и Println -- понимание их различий важно для написания читаемого кода.

Пакет fmt

Пакет fmt (сокращение от "format") -- стандартный пакет Go для работы с вводом и выводом. Чтобы его использовать, нужно импортировать:

Go 
import "fmt"

В этом пакете есть несколько функций для вывода. Начнем с двух базовых: Print и Println.

Print -- вывод без перевода строки

fmt.Print выводит данные в консоль и оставляет курсор на той же строке:

Go 
package main

import "fmt"

func main() {
    fmt.Print("Hello")
    fmt.Print("World")
}

Вывод:

HelloWorld

Слова "склеились", потому что после первого Print курсор остался на месте -- второй Print продолжил вывод в ту же строку.

Как добавить перенос строки вручную?

Используй специальный символ \n (обратный слэш + n):

Go 
fmt.Print("Hello\n")  // после Hello будет перенос
fmt.Print("World")

Вывод:

Hello
World
*
Tip

\n -- это escape-последовательность. Обратный слэш говорит: "следующий символ -- специальный". n означает "new line" (новая строка).

Println -- вывод с переводом строки

fmt.Println делает то же самое, что Print, но автоматически добавляет перенос строки в конце:

Go 
package main

import "fmt"

func main() {
    fmt.Println("Hello")
    fmt.Println("World")
}

Вывод:

Hello
World

Буква ln в названии -- сокращение от "line" (строка).

Главное отличие

Print -- выводит и оставляет курсор на месте
Println -- выводит и переводит курсор на новую строку

Визуально:

После Print("Hello"):
Hello_

курсор здесь

После Println("Hello"):
Hello
_

курсор здесь

Несколько аргументов

Обе функции принимают любое количество аргументов:

Go 
name := "Павел"
fmt.Println("Привет,", name)  // Привет, Павел

Вывод:

Привет, Павел

Важный нюанс: пробелы между аргументами

Println автоматически ставит пробел между аргументами:

Go 
fmt.Println("A", "B", "C")  // A B C

А Print -- не ставит:

Go 
fmt.Print("A", "B", "C")  // ABC

Это частая ловушка на собеседованиях!

Пустой Println

Println() без аргументов просто вставляет пустую строку:

Go 
fmt.Println("Строка 1")
fmt.Println()              // пустая строка
fmt.Println("Строка 3")

Вывод:

Строка 1

Строка 3

Вывод разных типов

Print и Println умеют выводить значения любых типов:

Go 
name := "Алиса"
age := 25
height := 1.65
active := true

fmt.Println(name)    // Алиса
fmt.Println(age)     // 25
fmt.Println(height)  // 1.65
fmt.Println(active)  // true

Go автоматически преобразует значения в текст для вывода.

Регистр имеет значение

!
Important

Go регистрозависимый. Println и println -- это разные вещи!

Go 
fmt.Println("OK")   // работает
fmt.println("OK")   // ошибка компиляции!

Функции пакета fmt начинаются с заглавной буквы (Print, Println). В Go заглавная буква в имени -- это правило: такие функции можно вызывать из других файлов. Подробнее об этом -- в модуле о пакетах.

Проверь себя

Вопросы

01 Чем отличается fmt.Print от fmt.Println?
Показать ответ

Print выводит данные и оставляет курсор на той же строке.

Println выводит данные и переводит курсор на новую строку.

02 Как называется пакет fmt полностью и что означает это сокращение?
Показать ответ

fmt -- сокращение от "format". Это стандартный пакет для форматирования, ввода и вывода данных.

03 Что выведет fmt.Println() без аргументов?
Показать ответ

Пустую строку. Это способ вставить вертикальный отступ между выводами.

04 Ставит ли Println пробелы между несколькими аргументами?
Показать ответ

Да, Println автоматически вставляет пробел между аргументами.

Print -- не вставляет.

Задачи

01 Что выведет?
Go 
fmt.Print("Go")
fmt.Print("lang")
Показать решение

Вывод: Golang

Почему: Print не добавляет перенос строки и не ставит пробелы. Второй вывод продолжается сразу после первого.

02 Что выведет?
Go 
fmt.Println("row1")
fmt.Print("row2")
fmt.Println("row3")
fmt.Print("row4")
Показать решение

Вывод:

row1
row2row3
row4

Почему:

  1. Println("row1") -- выводит "row1", переводит строку
  2. Print("row2") -- выводит "row2", курсор остается
  3. Println("row3") -- выводит "row3" сразу после "row2", переводит строку
  4. Print("row4") -- выводит "row4"
03 Что выведет?
Go 
fmt.Println("A", "B", "C")
fmt.Print("X", "Y", "Z")
Показать решение

Вывод:

A B C
XYZ

Почему: Println вставляет пробелы между аргументами, Print -- нет.

04 Что выведет?
Go 
fmt.Print("1\n2\n")
fmt.Println("3")
Показать решение

Вывод:

1
2
3

Почему: \n в Print создает переносы строк вручную. После "2" есть \n, затем Println("3") выводит "3" на новой строке.

05 Найди ошибку
Go 
package main

import "fmt"

func main() {
    fmt.println("Hello")
}
Показать решение

Ошибка: println написано с маленькой буквы.

Правильно: fmt.Println("Hello")

Go регистрозависимый. Функции пакета fmt начинаются с заглавной буквы.

Резюме

Key Print и Println
  • Print -- выводит данные, курсор остается на месте
  • Println -- выводит данные и переводит строку (ln = line)
  • Пробелы: Println вставляет между аргументами, Print -- нет
  • Перенос вручную: используй \n в строке
*
Tip

Когда что использовать: Println -- для большинства случаев (удобнее). Print -- когда нужен вывод в одну строку или полный контроль над форматом.
