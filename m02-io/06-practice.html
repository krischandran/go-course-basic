Задание: Генератор визитки

Напиши программу, которая:

  1. Спрашивает имя и фамилию
  2. Спрашивает должность
  3. Спрашивает компанию
  4. Спрашивает email и телефон
  5. Выводит красиво оформленную визитку

Решение

Полный код
Go 
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    scanner := bufio.NewScanner(os.Stdin)

    fmt.Println("=== Генератор визитки ===")
    fmt.Println()

    // Запрашиваем данные
    fmt.Print("Введите имя и фамилию: ")
    scanner.Scan()
    fullName := scanner.Text()

    fmt.Print("Введите должность: ")
    scanner.Scan()
    position := scanner.Text()

    fmt.Print("Введите компанию: ")
    scanner.Scan()
    company := scanner.Text()

    fmt.Print("Введите email: ")
    scanner.Scan()
    email := scanner.Text()

    fmt.Print("Введите телефон: ")
    scanner.Scan()
    phone := scanner.Text()

    // Выводим визитку
    fmt.Println()
    fmt.Println("+--------------------------------------+")
    fmt.Println("|          ВИЗИТНАЯ КАРТОЧКА           |")
    fmt.Println("+--------------------------------------+")
    fmt.Printf("|  %-36s|\n", fullName)
    fmt.Printf("|  %-36s|\n", position)
    fmt.Printf("|  %-36s|\n", company)
    fmt.Println("+--------------------------------------+")

    emailLine := fmt.Sprintf("Email: %s", email)
    phoneLine := fmt.Sprintf("Тел:   %s", phone)

    fmt.Printf("|  %-36s|\n", emailLine)
    fmt.Printf("|  %-36s|\n", phoneLine)
    fmt.Println("+--------------------------------------+")
}

Разбор решения

Почему везде Scanner?

В этой программе мы используем bufio.Scanner для всех полей, потому что:

  • Имя и фамилия содержат пробел
  • Должность может содержать пробел ("Go разработчик")
  • Компания может содержать пробел ("Яндекс Практикум")

Можно было бы использовать fmt.Scan для email и телефона (они без пробелов), но проще использовать один подход для всего.

Форматирование визитки

Go 
fmt.Printf("|  %-36s|\n", fullName)
  • %-36s -- строка шириной 36 символов, выравнивание влево
  • Минус - означает выравнивание влево
  • 36 -- ширина поля (чтобы визитка была ровной)
  • Пробелы справа заполняют до нужной ширины

Комбинирование Sprintf и Printf

Go 
emailLine := fmt.Sprintf("Email: %s", email)
fmt.Printf("|  %-36s|\n", emailLine)

Сначала собираем строку через Sprintf, потом выводим ее с форматированием через Printf. Это удобно, когда нужно сложное форматирование.

Дополнительные задания

01 Добавь дату

Добавь в визитку текущую дату. Для простоты -- пусть пользователь вводит ее сам.

Введите сегодняшнюю дату: 15.01.2024

fmt.Print("Введите сегодняшнюю дату: ")

// В выводе визитки:
dateLine := fmt.Sprintf("Дата: %s", date)

02 Выбор формата

Создай два варианта вывода: короткий и полный. Спроси пользователя, какой формат он хочет (1 или 2).

Короткий:

Анна Петрова | Go разработчик | anna@yandex.ru

Полный: (как в основном задании)

Показать решение
Go 
fmt.Print("Выберите формат (1-короткий, 2-полный): ")
var format int
fmt.Scan(&format)

// Короткий формат (format == 1)
shortCard := fmt.Sprintf("%s | %s | %s", fullName, position, email)
fmt.Println(shortCard)

// Полный формат (format == 2) -- как в основном решении
*
Tip

Как выбирать между вариантами (конструкция if) -- изучим в следующем модуле про условия. Пока можно просто вывести оба формата.

03 Номер визитки

Добавь порядковый номер визитки с ведущими нулями (например, #00042).

+--------------------------------------+
|  ВИЗИТКА #00042                      |
...
Показать решение
Go 
fmt.Print("Введите номер визитки: ")
var cardNumber int
fmt.Scan(&cardNumber)

// В выводе:
header := fmt.Sprintf("ВИЗИТКА #%05d", cardNumber)
fmt.Printf("|  %-36s|\n", header)

%05d -- число с ведущими нулями, ширина 5 символов.

Резюме модуля

Key Вывод
  • Print -- без перевода строки
  • Println -- с переводом строки, пробелы между аргументами
  • Printf -- по шаблону со спецификаторами
Key Ввод
  • Scan -- для слов и чисел (разделяет по пробелам)
  • bufio.Scanner -- для строк целиком
Key Форматирование
  • Sprintf -- собрать строку по шаблону
  • Errorf -- создать ошибку по шаблону
Key Основные спецификаторы
  • %s -- строка
  • %d -- целое число
  • %f -- дробное число (.2f -- 2 знака)
  • %v -- любое значение
  • %-10s -- строка шириной 10, влево
  • %05d -- число шириной 5, ведущие нули
