Printf и форматирование в Go

Простой вывод через Println быстро перестает хватать. Когда нужно вывести "Привет, Павел! Тебе 35 лет" -- склеивать строки через запятую неудобно. Для этого есть Printf.

Printf -- форматированный вывод

fmt.Printf выводит строку по шаблону, в который подставляются значения:

Go 
name := "Павел"
age := 35

fmt.Printf("Меня зовут %s, мне %d лет\n", name, age)

Вывод:

Меня зовут Павел, мне 35 лет

Как это работает?

Printf("шаблон", значение1, значение2, ...)
%s -- место для строки (string)
%d -- место для целого числа (decimal)

Спецификатор (format verb) -- это метка в строке, которая начинается с %. На её место подставляется значение.

!
Important

Printf не добавляет перенос строки в конце. Добавляй \n вручную, если нужен перенос.

Базовые спецификаторы

Это минимальный набор, который нужно знать:

Спецификатор Для чего Пример
%s Строка (string) "Павел"
%d Целое число (int) 35
%f Дробное число (float) 3.14
%t Булево значение (bool) true
%v Любое значение что угодно

%s -- строки

Go 
name := "Алиса"
fmt.Printf("Привет, %s!\n", name)  // Привет, Алиса!

%d -- целые числа

Go 
age := 25
fmt.Printf("Возраст: %d лет\n", age)  // Возраст: 25 лет

Буква d -- от слова "decimal" (десятичный).

%f -- дробные числа

Go 
price := 19.99
fmt.Printf("Цена: %f\n", price)  // Цена: 19.990000

По умолчанию выводится 6 знаков после точки. Если их меньше -- добьет нулями.

%t -- булевы значения

Go 
active := true
fmt.Printf("Активен: %t\n", active)  // Активен: true

%v -- универсальный

Go 
fmt.Printf("Значение: %v\n", 42)       // Значение: 42
fmt.Printf("Значение: %v\n", "текст")  // Значение: текст
fmt.Printf("Значение: %v\n", true)     // Значение: true

%v выводит значение в формате "по умолчанию" для его типа.

*
Tip

Когда использовать %v: Для отладки или когда тип не важен. Для продакшн-кода лучше конкретные спецификаторы -- они делают намерение явным.

Несколько спецификаторов

Значения подставляются по порядку:

Go 
name := "Павел"
surname := "Тарасов"
fmt.Printf("Привет, %s %s!\n", name, surname)
// Привет, Павел Тарасов!

Первый %s получает первое значение (name), второй %s -- второе значение (surname).

!
Important

Количество спецификаторов должно совпадать с количеством аргументов. Иначе -- ошибка или неожиданный результат.

Точность для дробных чисел

По умолчанию %f выводит 6 знаков. Часто это слишком много:

Go 
pi := 3.14159265
fmt.Printf("Пи: %f\n", pi)  // Пи: 3.141593 (6 знаков)

Чтобы указать количество знаков, используй точку с числом:

Go 
fmt.Printf("Пи: %.2f\n", pi)  // Пи: 3.14 (2 знака)
fmt.Printf("Пи: %.3f\n", pi)  // Пи: 3.142 (3 знака, округлено!)
fmt.Printf("Пи: %.0f\n", pi)  // Пи: 3 (0 знаков)
*
Tip

При указании точности число округляется по правилам математики. 3.14159 с .3f станет 3.142, потому что следующая цифра (5) >= 5.

Вывод символа процента

Знак % зарезервирован для спецификаторов. Чтобы вывести сам символ %, используй %%:

Go 
discount := 15
fmt.Printf("Скидка: %d%%\n", discount)  // Скидка: 15%

Sprintf -- форматирование в строку

Printf сразу выводит результат. А если нужно сохранить отформатированную строку?

Для этого есть Sprintf -- он возвращает строку, а не выводит:

Go 
name := "Павел"
age := 35

// Собираем строку
str := fmt.Sprintf("Меня зовут %s, мне %d лет", name, age)

// Теперь можем использовать str
fmt.Println(str)  // Меня зовут Павел, мне 35 лет

Когда использовать Sprintf?

  • Нужно сохранить строку в переменную
  • Нужно передать строку в другую функцию
  • Нужно склеить сложную строку для логирования
Go 
// Пример: создание сообщения об ошибке
msg := fmt.Sprintf("пользователь %s не найден", username)

Системы счисления (для любознательных)

Числа можно выводить в разных системах счисления:

Спецификатор Система Пример для 123
%d Десятичная 123
%b Двоичная 1111011
%o Восьмеричная 173
%x Шестнадцатеричная 7b
%X Шестнадцатеричная (заглавные) 7B
Go 
num := 123
fmt.Printf("Десятичная: %d\n", num)    // 123
fmt.Printf("Двоичная: %b\n", num)      // 1111011
fmt.Printf("Шестнадцатеричная: %x\n", num)  // 7b

С префиксом # добавляется обозначение системы:

Go 
fmt.Printf("Hex: %#x\n", 123)  // 0x7b

Где смотреть все спецификаторы?

Спецификаторов много, и не нужно их зубрить. Когда понадобится -- смотри документацию:

https://pkg.go.dev/fmt

Там описаны все спецификаторы: для указателей (%p), типов (%T), и многие другие.

Проверь себя

Вопросы

01 Чем отличается Printf от Println?
Показать ответ

Printf выводит по шаблону со спецификаторами (%s, %d и т.д.) и не добавляет перенос строки.

Println просто выводит значения через пробел и добавляет перенос строки.

02 Какой спецификатор использовать для вывода дробного числа с 2 знаками после точки?
Показать ответ

%.2f -- точка и число указывают количество знаков после запятой.

03 Чем отличается Printf от Sprintf?
Показать ответ

Printf выводит отформатированную строку в консоль.

Sprintf возвращает отформатированную строку (не выводит), её можно сохранить в переменную.

04 Как вывести символ % в строке форматирования?
Показать ответ

Использовать %%. Один % начинает спецификатор, два %% выводят сам символ процента.

Задачи

01 Что выведет?
Go 
fmt.Printf("Результат: %d", 10+5)
Показать решение

Вывод: Результат: 15

Почему: Выражение 10+5 вычисляется до подстановки, получается 15. %d для целого числа.

02 Что выведет?
Go 
fmt.Printf("%s + %s = %d\n", "5", "3", 8)
Показать решение

Вывод: 5 + 3 = 8

Почему: Первые два %s подставляют строки "5" и "3", %d подставляет число 8.

03 Что выведет?
Go 
fmt.Printf("Цена: %.2f руб.\n", 19.5)
Показать решение

Вывод: Цена: 19.50 руб.

Почему: .2f означает 2 знака после точки. Число 19.5 дополняется нулем до 19.50.

04 Что выведет?
Go 
fmt.Printf("Скидка %d%%\n", 25)
Показать решение

Вывод: Скидка 25%

Почему: %d подставляет число 25, %% выводит символ процента.

05 Найди ошибку
Go 
name := "Алиса"
age := 30
fmt.Printf("Имя: %s, возраст: %s\n", name, age)
Показать решение

Ошибка: Для age (число) использован %s (строка).

Правильно: fmt.Printf("Имя: %s, возраст: %d\n", name, age)

Хотя код может сработать (Go достаточно умен), это логическая ошибка. Для чисел используй %d.

Резюме

Key Printf и форматирование
  • Printf -- форматированный вывод по шаблону со спецификаторами
  • Базовые спецификаторы: %s -- строка, %d -- целое число, %f -- дробное (6 знаков), %t -- булево, %v -- универсальный
  • Точность: %.2f -- 2 знака после точки (с округлением)
  • Sprintf -- как Printf, но возвращает строку вместо вывода
  • %% -- вывести символ процента
*
Tip

Не зубри все спецификаторы. Запомни базовые (%s, %d, %f, %v), остальные смотри в документации по мере необходимости.
