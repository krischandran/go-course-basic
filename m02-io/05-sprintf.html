Sprintf и форматирование строк

В уроке про Printf мы уже познакомились с Sprintf. Теперь разберем его подробнее -- когда использовать и какие паттерны встречаются чаще всего.

Напоминание: Printf vs Sprintf

Функция Что делает Возвращает
Printf Выводит в консоль ничего (количество байт и ошибку)
Sprintf Собирает строку string
Go 
// Printf -- сразу на экран
fmt.Printf("Привет, %s!\n", name)

// Sprintf -- в переменную
greeting := fmt.Sprintf("Привет, %s!", name)
// greeting можно использовать дальше

Когда использовать Sprintf?

1. Сохранить результат для дальнейшего использования

Go 
// Нужно использовать строку несколько раз
username := "Алексей"
message := fmt.Sprintf("Пользователь %s вошел в систему", username)

fmt.Println(message)  // вывод на экран
// message можно передать куда угодно -- это обычная строка

2. Составление сложных строк

Go 
// Составляем путь к файлу
userID := "user123"
path := fmt.Sprintf("data/%s/config.json", userID)
fmt.Println(path)  // data/user123/config.json

// Составляем URL
id := 42
url := fmt.Sprintf("https://api.example.com/users/%d", id)
fmt.Println(url)  // https://api.example.com/users/42

3. Создать сообщение об ошибке

Go 
// fmt.Errorf -- как Sprintf, но возвращает error
orderID := 42
err := fmt.Errorf("заказ %d не найден", orderID)
fmt.Println(err)  // заказ 42 не найден
*
Tip

fmt.Errorf -- это как Sprintf, но возвращает error вместо string. Это стандартный способ создания ошибок с контекстом. Подробнее об обработке ошибок -- в модуле функций.

Errorf -- Sprintf для ошибок

fmt.Errorf работает как Sprintf, но создает значение типа error:

Go 
// Sprintf возвращает string
filename := "config.json"
msg := fmt.Sprintf("файл %s не найден", filename)
fmt.Println(msg)  // файл config.json не найден

// Errorf возвращает error
err := fmt.Errorf("файл %s не найден", filename)
fmt.Println(err)  // файл config.json не найден (но это error, не string)

Пример с переменными:

Go 
userID := 42
path := "/users/42/data"

// Создаем ошибку с контекстом
err := fmt.Errorf("не удалось загрузить данные пользователя %d по пути %s", userID, path)
fmt.Println(err)
// не удалось загрузить данные пользователя 42 по пути /users/42/data
*
Tip

Спецификатор %w позволяет "оборачивать" одну ошибку в другую. Это продвинутая тема -- подробнее в модуле про обработку ошибок.

Частые паттерны

Составление путей

Go 
// Путь к файлу пользователя
configPath := fmt.Sprintf("users/%d/settings.json", userID)

// URL для API запроса
url := fmt.Sprintf("https://api.example.com/users/%s/orders", username)

Форматирование для вывода

Go 
// Выравнивание в таблице
header := fmt.Sprintf("%-20s %10s %10s", "Название", "Цена", "Кол-во")
row := fmt.Sprintf("%-20s %10.2f %10d", "Ноутбук", 89990.50, 3)

fmt.Println(header)
fmt.Println(row)
// Название                  Цена     Кол-во
// Ноутбук               89990.50          3

Модификаторы ширины:

Формат Результат для "Go" Описание
%s Go как есть
%10s Go ширина 10, выравнивание вправо
%-10s Go ширина 10, выравнивание влево

Числа с ведущими нулями

Go 
// Номер заказа с ведущими нулями
orderNum := fmt.Sprintf("ORD-%06d", 42)  // "ORD-000042"

// Время
time := fmt.Sprintf("%02d:%02d:%02d", hours, minutes, seconds)  // "09:05:03"

Модификатор 0 добавляет ведущие нули вместо пробелов:

Формат Результат для 42 Описание
%d 42 как есть
%5d 42 ширина 5, пробелы слева
%05d 00042 ширина 5, нули слева

Денежные суммы

Go 
// Цена с двумя знаками после точки
price := fmt.Sprintf("%.2f руб.", 1234.5)  // "1234.50 руб."

// С разделителем тысяч (нужна библиотека или своя функция)
// Стандартный fmt этого не умеет

Sprintf vs конкатенация

Когда строк мало, можно использовать +:

Go 
// Простая конкатенация -- ОК
greeting := "Привет, " + name + "!"

// Но с числами нужно преобразование
// age -- это int, нельзя просто склеить
message := "Возраст: " + strconv.Itoa(age) + " лет"  // громоздко

// Sprintf проще
message := fmt.Sprintf("Возраст: %d лет", age)

Когда что использовать:

Ситуация Рекомендация
2-3 строки без чисел + конкатенация
Есть числа или форматирование Sprintf
Много частей Sprintf
*
Tip

Для сборки строк по частям в цикле есть эффективный strings.Builder. Подробнее -- в модуле про строки и циклы.

Проверь себя

Вопросы

01 Чем Sprintf отличается от Printf?
Показать ответ

Printf выводит отформатированную строку в консоль. Sprintf возвращает отформатированную строку, которую можно сохранить в переменную или передать дальше.

02 Что делает fmt.Errorf и чем отличается от Sprintf?
Показать ответ

fmt.Errorf работает как Sprintf, но возвращает значение типа error вместо string. Это стандартный способ создания ошибок с контекстом.

03 Как вывести число с ведущими нулями, например 42 как "00042"?
Показать ответ

Использовать модификатор 0 перед шириной: %05d. Это означает "ширина 5 символов, заполнить нулями слева".

Go 
fmt.Sprintf("%05d", 42)  // "00042"
04 Когда лучше использовать конкатенацию (+), а когда Sprintf?
Показать ответ

Конкатенация (+) -- для 2-3 строк без чисел. Sprintf -- когда есть числа, нужно форматирование, или много частей.

Задачи

01 Что вернет?
Go 
result := fmt.Sprintf("ID: %05d", 7)
fmt.Println(result)
Показать решение

Вывод: ID: 00007

Почему: %05d означает "целое число шириной 5 символов с ведущими нулями". Число 7 дополняется четырьмя нулями слева.

02 Что вернет?
Go 
name := "Go"
result := fmt.Sprintf("[%-10s]", name)
fmt.Println(result)
Показать решение

Вывод: [Go ]

Почему: %-10s означает "строка шириной 10 символов, выравнивание влево". "Go" занимает 2 символа, остальные 8 заполняются пробелами справа.

03 Что вернет?
Go 
price := 9.5
result := fmt.Sprintf("Цена: %.2f", price)
fmt.Println(result)
Показать решение

Вывод: Цена: 9.50

Почему: %.2f означает "дробное число с 2 знаками после точки". Число 9.5 дополняется нулем до 9.50.

04 Составь путь

Напиши код, который создает путь к файлу в формате users/<id>/avatar.png, где id -- переменная типа int.

Go 
id := 42
// Твой код
Показать решение
Go 
id := 42
path := fmt.Sprintf("users/%d/avatar.png", id)
// path = "users/42/avatar.png"
05 Форматирование времени

Напиши код, который форматирует время в виде "09:05:03" (с ведущими нулями).

Go 
hours := 9
minutes := 5
seconds := 3
// Твой код
Показать решение
Go 
hours := 9
minutes := 5
seconds := 3
time := fmt.Sprintf("%02d:%02d:%02d", hours, minutes, seconds)
// time = "09:05:03"

Почему: %02d для каждого числа обеспечивает ширину 2 символа с ведущим нулем при необходимости.

Резюме

Key Sprintf и форматирование
  • Sprintf -- форматирование в строку (не на экран)
  • Errorf -- как Sprintf, но возвращает error
  • %10s -- ширина 10, вправо
  • %-10s -- ширина 10, влево
  • %05d -- ширина 5, ведущие нули
  • Для простой склейки 2-3 строк без чисел -- используй +
  • Когда есть числа или нужно форматирование -- Sprintf
