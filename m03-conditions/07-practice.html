Упражнение: практика условий

12 задач для закрепления всех конструкций условного ветвления: if-else, else if, switch, логические операторы.

Что мы изучили в этом модуле

В модуле "Условия" мы научились:

  • Использовать if и else для ветвления программы
  • Применять операторы сравнения (>, <, >=, <=, ==, !=)
  • Комбинировать условия с помощью &&, ||, !
  • Строить цепочки else if для множественных условий
  • Использовать switch-case для сравнения с конкретными значениями
  • Применять switch без выражения для проверки диапазонов

Теперь закрепим знания на практике.

Задачи

01 Доступ к системе

Напиши программу, которая проверяет, разрешён ли пользователю доступ к системе.

Условие: Доступ разрешён, если возраст не менее 12 лет.

Вход: одно целое число — возраст пользователя.

Выход:

  • Доступ разрешен если возраст >= 12
  • Доступ запрещен иначе
Примеры
Вход: 10 → Выход: Доступ запрещен
Вход: 15 → Выход: Доступ разрешен
Показать решение
Go 
package main

import "fmt"

func main() {
    var age int
    fmt.Scan(&age)

    if age >= 12 {
        fmt.Println("Доступ разрешен")
    } else {
        fmt.Println("Доступ запрещен")
    }
}
02 Максимум из двух

Напиши программу, которая находит максимальное из двух целых чисел. Если числа равны — выведи любое из них.

Вход: два целых числа.

Выход: максимальное число.

Примеры
Вход: 6 5 → Выход: 6
Вход: 3 3 → Выход: 3
Показать решение
Go 
package main

import "fmt"

func main() {
    var a, b int
    fmt.Scan(&a, &b)

    if a >= b {
        fmt.Println(a)
    } else {
        fmt.Println(b)
    }
}

05 Трёхзначное число

Напиши программу, которая определяет, является ли число трёхзначным.

Вход: одно натуральное число.

Выход: YES если трёхзначное (от 100 до 999), NO иначе.

Примеры
Вход: 300 → Выход: YES

package main

06 Принадлежность интервалу

Напиши программу, которая проверяет, принадлежит ли число x интервалу (-1, 17) — не включая границы.

Вход: одно целое число.

Выход: Принадлежит или Не принадлежит.

Примеры
Вход: 2 → Выход: Принадлежит
Вход: -1 → Выход: Не принадлежит
Вход: 17 → Выход: Не принадлежит
Показать решение
Go 
package main

import "fmt"

func main() {
    var x int
    fmt.Scan(&x)

    if x > -1 && x < 17 {
        fmt.Println("Принадлежит")
    } else {
        fmt.Println("Не принадлежит")
    }
}
07 Различные цифры

По данному трёхзначному числу определи, все ли его цифры различны.

Вход: одно трёхзначное число.

Выход: YES если все цифры разные, NO если есть повторяющиеся.

Примеры
Вход: 125 → Выход: YES
Вход: 121 → Выход: NO
Показать решение
Go 
package main

import "fmt"

func main() {
    var n int
    fmt.Scan(&n)

    d1 := n / 100       // первая цифра
    d2 := (n / 10) % 10 // вторая цифра
    d3 := n % 10        // третья цифра

    if d1 != d2 && d2 != d3 && d1 != d3 {
        fmt.Println("YES")
    } else {
        fmt.Println("NO")
    }
}

Объяснение:

  • n / 100 — целочисленное деление даёт первую цифру (125 / 100 = 1)
  • (n / 10) % 10 — средняя цифра (125 / 10 = 12, 12 % 10 = 2)
  • n % 10 — последняя цифра (125 % 10 = 5)
08 Знак числа (sign)

В математике функция sign(x) определена так:

  • -1 если x < 0
  • 0 если x = 0
  • 1 если x > 0

Вход: одно целое число.

Выход: значение sign(x).

Примеры
Вход: 15 → Выход: 1
Вход: -7 → Выход: -1
Вход: 0 → Выход: 0
Показать решение
Go 
package main

import "fmt"

func main() {
    var x int
    fmt.Scan(&x)

    if x > 0 {
        fmt.Println(1)
    } else if x < 0 {
        fmt.Println(-1)
    } else {
        fmt.Println(0)
    }
}
09 Возрастная группа

Напиши программу, которая по возрасту определяет возрастную группу:

  • до 13 включительно → детство
  • от 14 до 24 включительно → молодость
  • от 25 до 59 включительно → зрелость
  • от 60 и старше → старость

Вход: одно натуральное число — возраст.

Выход: название возрастной группы.

Примеры
Вход: 4 → Выход: детство
Вход: 30 → Выход: зрелость
Показать решение
Go 
package main

import "fmt"

func main() {
    var age int
    fmt.Scan(&age)

    if age <= 13 {
        fmt.Println("детство")
    } else if age <= 24 {
        fmt.Println("молодость")
    } else if age <= 59 {
        fmt.Println("зрелость")
    } else {
        fmt.Println("старость")
    }
}
10 Время года

По номеру месяца (1-12) определи время года.

Вход: целое число от 1 до 12.

Выход: Зима, Весна, Лето или Осень.

Примеры
Вход: 4 → Выход: Весна
Вход: 12 → Выход: Зима
Показать решение
Go 
package main

import "fmt"

func main() {
    var month int
    fmt.Scan(&month)

    switch month {
    case 12, 1, 2:
        fmt.Println("Зима")
    case 3, 4, 5:
        fmt.Println("Весна")
    case 6, 7, 8:
        fmt.Println("Лето")
    case 9, 10, 11:
        fmt.Println("Осень")
    }
}
11 Високосный год

Год является високосным, если:

  • Делится на 4, но НЕ делится на 100
  • ИЛИ делится на 400

Вход: натуральное число — год.

Выход: YES если год високосный, NO иначе.

Примеры
Вход: 2020 → Выход: YES
Вход: 1900 → Выход: NO
Вход: 2000 → Выход: YES
Показать решение
Go 
package main

import "fmt"

func main() {
    var year int
    fmt.Scan(&year)

    if (year%4 == 0 && year%100 != 0) || year%400 == 0 {
        fmt.Println("YES")
    } else {
        fmt.Println("NO")
    }
}

Объяснение:

  • 2020: делится на 4 (2020%4=0), не делится на 100 (2020%100=20) → високосный
  • 1900: делится на 4, делится на 100, не делится на 400 → не високосный
  • 2000: делится на 400 → високосный
12 Ладья

Ладья ходит по горизонтали или вертикали. По координатам ладьи и другой фигуры определи, может ли ладья побить фигуру за один ход.

Вход: четыре числа — координаты ладьи (строка, столбец) и координаты фигуры (строка, столбец). Каждое число от 1 до 8.

Выход: YES если ладья может побить фигуру, NO иначе.

Примеры
Вход: 1 1 1 5 → Выход: YES
Вход: 1 1 2 5 → Выход: NO
Показать решение
Go 
package main

import "fmt"

func main() {
    var r1, c1, r2, c2 int
    fmt.Scan(&r1, &c1, &r2, &c2)

    if r1 == r2 || c1 == c2 {
        fmt.Println("YES")
    } else {
        fmt.Println("NO")
    }
}

Объяснение: Ладья бьёт фигуру, если они на одной строке (r1 == r2) ИЛИ на одном столбце (c1 == c2).
