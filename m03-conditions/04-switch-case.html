Оператор switch

Элегантный способ сравнить переменную с множеством конкретных значений.

SWITCH НА САМОМ ДЕЛЕ

Когда нужно сравнить переменную с множеством конкретных значений, цепочка if-else if становится громоздкой:

Go
// Громоздко при большом количестве вариантов
if action == "Прыгать" {
    fmt.Println("Мне нравится прыгать")
} else if action == "Плавать" {
    fmt.Println("Я люблю плавать")
} else if action == "Летать" {
    fmt.Println("Хотел бы научиться летать")
}

Для таких случаев в Go есть оператор switch — он делает код компактнее и читабельнее.

БАЗОВЫЙ СИНТАКСИС SWITCH

Go
package main

import "fmt"

func main() {
    var action string
    fmt.Scan(&action)

    switch action {
    case "Прыгать":
        fmt.Println("Мне нравится прыгать")
    case "Плавать":
        fmt.Println("Я люблю плавать")
    case "Летать":
        fmt.Println("Хотел бы научиться летать")
    }
}

Как это работает:

  1. Значение action сравнивается с каждым case по порядку
  2. Как только найдено совпадение — выполняется соответствующий блок
  3. После выполнения блока switch завершается (без проваливания в следующие case)
  4. Если ни один case не совпал — ничего не происходит

БЛОК DEFAULT

Чтобы обработать случай, когда значение не совпало ни с одним case, используется default:

Go
package main

import "fmt"

func main() {
    var action string
    fmt.Scan(&action)

    switch action {
    case "Прыгать":
        fmt.Println("Мне нравится прыгать")
    case "Плавать":
        fmt.Println("Я люблю плавать")
    case "Летать":
        fmt.Println("Хотел бы научиться летать")
    default:
        fmt.Println("Могу только ходить")
    }
}

При вводе "Бегать" ни один case не совпадёт, выполнится default.

+

default — аналог финального else в цепочке if-else if-else.

ОБЪЕДИНЕНИЕ НЕСКОЛЬКИХ CASE

Если несколько значений должны обрабатываться одинаково, их можно объединить через запятую:

Go
package main

import "fmt"

func main() {
    var day string
    fmt.Scan(&day)

    switch day {
    case "Понедельник", "Вторник", "Среда", "Четверг", "Пятница":
        fmt.Println("Будний день")
    case "Суббота", "Воскресенье":
        fmt.Println("Выходной день")
    default:
        fmt.Println("Некорректный день недели")
    }
}

Работает так: если day равен любому из значений в case, выполняется этот блок.

SWITCH С ЧИСЛАМИ

switch работает не только со строками:

Go
package main

import "fmt"

func main() {
    var month int
    fmt.Scan(&month)

    switch month {
    case 1, 2, 12:
        fmt.Println("Зима")
    case 3, 4, 5:
        fmt.Println("Весна")
    case 6, 7, 8:
        fmt.Println("Лето")
    case 9, 10, 11:
        fmt.Println("Осень")
    default:
        fmt.Println("Некорректный месяц")
    }
}

При вводе 4:

  • Проверяем 4 в 1, 2, 12? Нет
  • Проверяем 4 в 3, 4, 5? Да -> выводим "Весна"

Когда использовать switch?

switch хорош когда:

  • Сравниваешь переменную с фиксированным набором значений
  • Много вариантов (3+)
  • Значения — константы (числа, строки)

if-else лучше когда:

  • Проверяешь диапазоны (x > 10 && x < 20)
  • Сложные логические выражения
  • Условия зависят от разных переменных

Ловушка: switch не работает с диапазонами напрямую

Go
// ОШИБКА: нельзя использовать выражения в case
var score int
fmt.Scan(&score)

switch score {
case score >= 90:  // Не скомпилируется!
    fmt.Println("Отлично")
}

В обычном switch после case должны быть конкретные значения, а не логические выражения.

Решение: используй if-else для диапазонов или switch без выражения (следующий урок).

ПРОВЕРЬ СЕБЯ

Вопросы

01Что произойдёт, если ни один case не совпал и нет default?
Показать ответ

Ничего не произойдёт. switch просто завершится без выполнения какого-либо кода.

02Можно ли объединить несколько значений в одном case?
Показать ответ

Да, значения перечисляются через запятую:

Go
case "a", "b", "c":
    // код
03Для чего нужен блок default?
Показать ответ

default выполняется, когда значение не совпало ни с одним case. Это аналог финального else — обрабатывает все оставшиеся случаи.

04Можно ли в case использовать условие вида x > 10?
Показать ответ

В обычном switch — нет, только конкретные значения. Для проверки диапазонов используй if-else или switch без выражения (будет в следующем уроке).

01Количество дней в месяце

По номеру месяца (1-12) выведи количество дней. Считай, что февраль имеет 28 дней.

Показать решение
Go
package main

import "fmt"

func main() {
    var month int
    fmt.Scan(&month)

    switch month {
    case 1, 3, 5, 7, 8, 10, 12:
        fmt.Println(31)
    case 4, 6, 9, 11:
        fmt.Println(30)
    case 2:
        fmt.Println(28)
    default:
        fmt.Println("Некорректный месяц")
    }
}

03Что выведет?
Go
package main

import "fmt"

func main() {
    day := "Среда"

    switch day {
    case "Понедельник":
        fmt.Print("1")
    case "Вторник", "Среда":
        fmt.Print("2")
    case "Четверг":
        fmt.Print("3")
    default:
        fmt.Print("4")
    }

    fmt.Print("!")
}
Показать решение

Вывод: 2!

Почему:

  1. day = "Среда"
  2. Проверяем "Понедельник"? Нет
  3. Проверяем "Вторник", "Среда"? Да (Среда в списке) -> выводим 2
  4. switch завершается
  5. Выводим !

РЕЗЮМЕ

KeyКлючевые идеи
  • switch сравнивает значение с набором case
  • Как только case совпал — выполняется блок и switch завершается
  • default обрабатывает все несовпавшие случаи
  • Несколько значений в case перечисляются через запятую

Главное ограничение: в обычном switch нельзя использовать диапазоны и сложные условия

Правило: switch для фиксированных значений, if-else для диапазонов и сложных условий
